# Activity Tracker

A simple activity tracker web application that captures text input with timestamps and stores it in a CSV file.

## Features

- Records activities with automatic date and time stamps
- Displays activities in a clean, sortable list
- Simple and responsive Bootstrap UI
- Data stored in CSV format for easy access and portability

## Technical Details

- Built with Flask (Python)
- Frontend using Bootstrap with dark theme
- Data stored in CSV file format
- JavaScript for dynamic content updates

## How to Run

1. Clone this repository
2. Install dependencies: `pip install flask gunicorn`
3. Run the application: `gunicorn --bind 0.0.0.0:5000 main:app`
4. Open your browser to http://localhost:5000
